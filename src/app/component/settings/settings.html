Display Settings

Enable Shapes Mode

When enabled, color-coded groups will display a unique shape in each cell, providing an alternative visual cue for players who may have difficulty distinguishing colors.

Show Row / Column Current Sum

Controls whether row and column headers display in the upper-right corner the current sum of selected cells.

Show Color Group Current Sum

Controls whether the sum of selected cells in a color group displays in the color groups first cell's upper-right corner.

Show Scratch Pad

Controls whether the scratch pad is displayed next to the board. More info about the scratch pad can be found in the.

Enable Section Completion Animations

Plays a cascading glow across rows, columns, or groups when they meet their target value but still have open cells, highlighting the progress you've made.

Auto Clear Cells

Unusable cells in completed sections will be automatically cleared and removed.
